# 🏥 Hospital Emergency Room Dashboard (Excel)

## 📌 Project Overview

This project is an interactive Hospital Emergency Room Dashboard developed in Microsoft Excel to analyze emergency room patient data. The dashboard provides healthcare administrators with valuable insights into patient admissions, waiting times, referrals, attendance performance, gender distribution, and age group analysis.

The dashboard updates dynamically through slicers, allowing users to filter the report by Month and Year.
